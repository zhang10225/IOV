# IOV

Internet of Vehicles (车联网) — 追责与撤销机制研究

## 项目说明

本项目聚焦车联网（IoV/VANET）场景下的**追责（Accountability/Traceability）**与**撤销（Revocation）**机制研究，涵盖近五年（2021–2026）的学术论文综述、可复现实验清单与改进路径。

## 文档

- [文献综述与复现实验清单](docs/literature_review.md) — 包含 10 篇核心论文的实验设置、指标、复现难点、改进方案及可执行复现清单。

## 研究方向

- 条件隐私保护认证（Conditional Privacy-Preserving Authentication）
- 群签名与追踪（Group Signature with Traceability）
- 区块链辅助追责与撤销（Blockchain-Based Accountability & Revocation）
- 证书/凭证撤销机制（Certificate/Credential Revocation）
- 零知识证明匿名认证（ZKP-Based Anonymous Authentication）
- 联邦学习异常检测（Federated Learning for Misbehavior Detection）
